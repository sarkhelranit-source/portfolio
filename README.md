## 🚀 Features
- **Immersive 3D Experience:** Powered by Spline & Three.js / React Three Fiber.
- **Dynamic Interactions:** Framer Motion for smooth, highly polished UI animations.
- **Responsive Design:** Fully tailored for mobile, tablet, and desktop viewing.
- **Modern Stack:** Built with React, Vite, and Tailwind CSS.

- **Frontend:** React.js, TypeScript, Tailwind CSS
- **Animations & 3D:** Framer Motion, Spline, Three.js, React Three Fiber
- **Build Tool:** Vite

## 🏃‍♂️ Getting Started
To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/sarkhelranit-source/portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
